parallel

好用的批处理工具

parallel 将 command 中的命令执行 n 次，每次赋予 text 中的不同值。

注意: parallel 是将每条命令执行多次，如果有判断后写入语句建议先单独执行一次，将内容写入后用 parallel 执行。
