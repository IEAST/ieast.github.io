功能

根据特定版本的分支创建新分支

git rebase

提示

需要在添加 upstream 的前提下使用。

使用方法

  1. git switch 切换到需要 rebase 的分支。
  2. git fetch upstream 将 upstream 的最新分支拉到本地。
  3. git rebase upstream/master rebase分支。(这时候分支已经干净了)
  4. git push -f 提交分支。
